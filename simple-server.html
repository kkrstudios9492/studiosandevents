üîß Solution: Start a Local Server

Choose one of these methods:

Method 1: Using Python (Easiest)

1. Open Command Prompt 2. Navigate to your project folder: cd "D:\kkr project" 3. Run: python -m http.server 8000 4. Open: http://localhost:8000/login.html

Method 2: Using Node.js

1. Open Command Prompt 2. Navigate to your project folder: cd "D:\kkr project" 3. Run: npx serve . 4. Open the URL it shows (usually http://localhost:3000)

Method 3: Using the Batch File

1. Double-click "start-local-server.bat" in your project folder 2. Open: http://localhost:8000/login.html

üîç Still Not Working?

If you still have issues after starting the server:

  1. Make sure you're using http://localhost:8000 (not file://)
  2. Check that your Google Cloud Console has http://localhost:8000 in authorized origins
  3. Try a different browser or incognito mode
  4. Disable any ad blockers temporarily

üìù Google Cloud Console Setup

If you haven't set up Google Cloud Console yet:

1. Go to: https://console.cloud.google.com/apis/credentials 2. Click on your OAuth 2.0 Client ID 3. Add to "Authorized JavaScript origins": - http://localhost:8000 - http://localhost:3000 4. Add to "Authorized redirect URIs": - http://localhost:8000/login.html - http://localhost:3000/login.html 5. Save changes
