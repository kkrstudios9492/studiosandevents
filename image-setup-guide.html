Step 3: Add Your Images

To replace the product images:

  1. Open the folder: D:\kkr project\images\products\
  2. Add your image files with these exact names:
    • premium-decorations.jpg
    • photography-package.jpg
    • venue-rental.jpg
    • birthday-party.jpg
    • wedding-photography.jpg
    • corporate-event.jpg
  3. Keep the same filenames - the code references these specific names

Step 4: Image Requirements

Recommended specifications:
  • Format: JPG or PNG
  • Size: 400x300 pixels (or similar aspect ratio)
  • File Size: Under 500KB per image
  • Quality: High quality, clear images

Step 5: Test Your Images

After adding your images:

  1. Open ecommerce.html in your browser
  2. Check that all product images load correctly
  3. If an image doesn't load, check the filename matches exactly

Step 6: Using Different Filenames (Advanced)

If you want to use different filenames:

  1. Open ecommerce.js
  2. Find the products array (around line 5)
  3. Update the image property for each product
  4. Example: image: 'images/products/your-custom-name.jpg'
